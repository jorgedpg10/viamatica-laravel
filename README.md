

## Información
El presente test se desarrolló utilizando la librería Saptie/ Query builder en la versión 4.

para ejecutar el API es necesario configurar la base de datos y ejecutar:  
 `composer install `  
 `php artisan migrate`  
`php artisan serve`

para facilitar la prueba del API se agregaron seeders en la clase Database\seeders\DatabaseSeeder el cual se puede ejecutar con:  
 
 `php artisan db:seed`
